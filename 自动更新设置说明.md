# 自动更新设置说明

本文档说明如何设置每周一自动检查 OpenClaw 官方文档更新。

---

## 方法一：使用 OpenClaw Cron（推荐）

### 前置要求

- OpenClaw Gateway 已运行并配对
- 拥有 cron 工具权限

### 设置步骤

1. 确保 Gateway 正在运行：
   ```bash
   openclaw gateway status
   ```

2. 使用以下 cron 配置：

```json
{
  "name": "OpenClaw文档每周更新检查",
  "schedule": {
    "kind": "cron",
    "expr": "0 0 * * 1",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "检查 OpenClaw 官方文档是否有更新，并更新中文教程文档。仓库位置：/home/node/.openclaw/workspace/projects/github/fx/",
    "model": "ark-coding-plan/doubao-seed-2.0-code",
    "timeoutSeconds": 600
  },
  "sessionTarget": "isolated",
  "enabled": true
}
```

3. 通过 OpenClaw Control UI 或命令行添加 cron 任务

---

## 方法二：使用系统 Cron

如果 OpenClaw cron 不可用，可以使用系统 cron：

### 设置步骤

1. 编辑 crontab：
   ```bash
   crontab -e
   ```

2. 添加以下行（每周一凌晨 00:00 运行）：
   ```bash
   0 0 * * 1 cd /home/node/.openclaw/workspace/projects/github/fx && ./check-update.sh >> /var/log/openclaw-doc-update.log 2>&1
   ```

3. 保存并退出

---

## 方法三：手动运行

随时可以手动运行更新检查：

```bash
cd /home/node/.openclaw/workspace/projects/github/fx
./check-update.sh
```

---

## 检查脚本说明

`check-update.sh` 脚本会：

1. 检查 git 状态
2. 拉取远程更新
3. 记录检查时间到更新日志
4. 更新教程文档中的最后更新时间
5. 提交并推送更改（如果有）

---

## Cron 表达式说明

`0 0 * * 1` 表示：
- 第 1 个 `0`：分钟（0分）
- 第 2 个 `0`：小时（0点，即午夜）
- 第 3 个 `*`：每月的任意一天
- 第 4 个 `*`：任意月份
- 第 5 个 `1`：星期一（1=周一，0=周日）

其他示例：
- `0 2 * * 1`：每周一凌晨 2 点
- `0 0 * * 0`：每周日午夜
- `0 */6 * * *`：每 6 小时
